# 🛠️ LearnTrackYT - Comprehensive Requirements & Setup Guide

This file contains all the packages, libraries, and commands needed to run **LearnTrackYT** locally. 

> [!TIP]
> **TO AUTOMATICALLY DOWNLOAD EVERYTHING:** Run the provided `setup.bat` file at the root of the project. It will handle the entire installation for you!

---

## 🚀 1. Complete Installation Guide

### Step 1: Install Global Tools
Ensure these are installed on your system:
- **Node.js (v20+)**: https://nodejs.org/
- **MongoDB**: https://www.mongodb.com/try/download/community
- **Deno**: Run this in PowerShell: `irm https://deno.land/install.ps1 | iex`

### Step 2: Running the Automated Setup
Double-click the **`setup.bat`** file in this folder. It will:
1. Check for Node.js.
2. Install Deno (if missing).
3. Install all Backend dependencies.
4. Install all Frontend dependencies.
5. Download `yt-dlp.exe` to the `bin/` folder.

---

## 📦 2. Full Package List

### 🖥️ Backend Packages (`/backend`)
*Install manually with:* `cd backend && npm install`
- **Core**: `express`, `cors`, `dotenv`, `helmet`, `morgan`
- **Database**: `mongoose`
- **AI/ML**: `groq-sdk`, `openai`
- **YouTube**: `youtubei.js`, `youtube-transcript`, `ytpl`
- **Security**: `jsonwebtoken`, `bcryptjs`
- **Utils**: `axios`, `fs`

### 🌐 Frontend Packages (`/frontend`)
*Install manually with:* `cd frontend && npm install`
- **Core**: `react`, `react-dom`, `vite`
- **Routing**: `react-router-dom`
- **Styling**: `tailwindcss`, `autoprefixer`, `postcss`, `tailwind-merge`, `clsx`
- **Components**: `framer-motion`, `lucide-react`, `recharts`, `react-youtube`
- **Mobile**: `@capacitor/core`, `@capacitor/cli`, `@capacitor/android`
- **Utils**: `axios`, `date-fns`, `jspdf`

---

## ⚡ 3. Running the Project
Once installed, use these commands:

| Service | Command | Directory |
| :--- | :--- | :--- |
| **Backend** | `npm start` | `/backend` |
| **Frontend** | `npm run dev` | `/frontend` |

---

## 🔑 4. Required Config (.env)
Create a `.env` file in the **`backend/`** folder:
```env
PORT=5000
MONGO_URI=mongodb://localhost:27017/learntrack
JWT_SECRET=your_secret_string
GROQ_API_KEY=your_groq_api_key
```

---

## 🏁 5. Troubleshooting
- **Deno Path**: If the server fails to find Deno, restart your PC or terminal.
- **yt-dlp**: Ensure `bin/yt-dlp.exe` exists (the setup script handles this).
- **MongoDB**: Make sure the MongoDB service is running on your machine.
